## What is it?

Package provides simple Gulp configuration with SASS compilation and browser auto refresh.

## Getting started

Repository setup:

* `node` + `npm`
* `gulp` (with `gulp-sass`)
* `browser-sync`

### First start

Install dependencies, running `npm install`.


### Development

Just run `gulp` or `gulp dev` and edit `.scss` / `.html` files.
